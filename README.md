# More-Recipes

Platform for users to share the awesome and exciting recipe ideas they have invented or learnt.

### Template Built With

 - Materialize CSS

### Server Built With

 - Node JS
 - Express

### Test Framework
 - Mocha
 - Chai
 
### Site URL

https://gbengaoshinaga.github.io/more-recipes/template/

### Server Side
 - Clone repository
  ```
  git clone https://github.com/GbengaOshinaga/more-recipes
  ```
 - Install dependecies
  ```
  npm install
  ```
 - Run the server
  ```
  npm run start
  ```
  - Run tests
  ```
  npm run test
  ```
